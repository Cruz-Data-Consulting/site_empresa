## Como publicar no GitHub Pages

### Opção 1 — Repositório novo só para o portfólio

1. Crie um novo repositório no GitHub.
   - Exemplo: `portfolio-felipe-cruz`
2. Envie os arquivos deste projeto para a raiz do repositório.
3. No GitHub, acesse:
   - `Settings` → `Pages`
4. Em **Build and deployment**:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main`
   - **Folder**: `/ (root)`
5. Salve.
6. Aguarde alguns minutos. O GitHub vai gerar a URL pública do site.

Exemplo de resultado:

```text
https://SEU-USUARIO.github.io/portfolio-felipe-cruz/
```

### Opção 2 — Publicar como site principal do perfil

Se quiser que o site abra diretamente no domínio principal do GitHub Pages, crie um repositório com este nome:

```text
F-Cruz.github.io
```

Depois:
1. Envie os arquivos para esse repositório.
2. Ative o GitHub Pages em `Settings` → `Pages`.
3. O site ficará disponível em:

```text
https://f-cruz.github.io/
```
